Data Products
=============

Current Published Artifacts
---------------------------

The runtime publish step currently uploads:

- PONI calibration file
- Per-scan azimuthal CSV outputs

Metadata and Provenance
-----------------------

Published artifacts include metadata linkage for:

- Workflow version
- Run ID
- Model linkage
- Calibrant linkage
- PONI geometry linkage
- Optional per-scan IGSN metadata when available

Current Non-Published Outputs
-----------------------------

- Lattice parameter outputs are computed by implementation modules but are not currently published in the active runtime path.
- A run manifest is not currently uploaded in the active runtime path.
